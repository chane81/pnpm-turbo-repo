# Pnpm + Turborepo + Docker

## Stack

- Pnpm: v8.7.6
- Turborepo: v1.10.14
- Docker
- Next.js: v13.4.19

## Command

```bash
# docker build - web
$ pnpm docker-build:web

# docker run - web
$ pnpm docker-run:web

# docker build - docs
$ pnpm docker-build:docs

# docker run - docs
$ pnpm docker-run:docs
```
